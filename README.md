🌐 Personal Portfolio Website

A responsive and interactive personal portfolio website built with HTML, CSS, and JavaScript. This project showcases skills, projects, and resume in a modern, user-friendly way.

🌟 Features

Home Page: Clean landing page with name, bio, and profile picture.

About Page: Detailed section with education, experience, and hobbies.

Projects Page: Gallery of projects with titles, descriptions, and images.

Contact Page: Contact form with JavaScript validation (Name, Email, Subject, Message).

Resume Download: Link to a downloadable PDF resume.

Responsive Design: Works on desktop, tablet, and mobile screens.

Smooth Navigation: Smooth scrolling and active link highlighting.

Interactivity: Hover effects, animations, and dynamic UI with JavaScript.

🛠 Technologies Used

HTML5 – Structure and semantic elements

CSS3 – Styling, responsive layouts, hover effects

JavaScript (ES6) – Interactivity, form validation, smooth scrolling
